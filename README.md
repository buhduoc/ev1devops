# Que es el README.md?

El README es uno de los documentos mas importantes dentro del proyecto, explica la
instruciones del proyecto y su proposito.

## Proposito del README

Como ya se habia dicho el README muestra datos del proyecto, este ocupa **Markdown** para su escritura.

## Solo Existe el README?

No solamente existe el readme, existe CONTRIBUTING, SECURITY, CHANGELOG, LICENSE, CODE_OF_CONhDUCT. Todos estan escritos en **Markdown**.
